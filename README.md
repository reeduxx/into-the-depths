# Installation & Running
## Requirements
- Python 3.x
- `blessed` module

## Steps
### Step 1: Clone the repository
```
git clone https://github.com/reeduxx/into-the-depths.git
cd into-the-depths
```

### Step 2: Install dependencies
```
pip install -r requirements.txt
```

### Step 3: Run the game
```
py src/main.py
```
